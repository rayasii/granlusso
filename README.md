# Granlusso Vue + Directus Full

## Setup

1. Clone repo:
   ```
   git clone <your repo>
   cd granlusso-vue-full
   ```
2. Install:
   ```
   npm install
   ```
3. Env:
   ```
   cp .env.example .env
   ```
4. Run:
   ```
   npm run dev
   ```
5. Build:
   ```
   npm run build
   ```
6. Deploy to Render:
   - Build: `npm install && npm run build`
   - Start: `npm run preview`
   - Envs: `VITE_DIRECTUS_URL`, `VITE_DIRECTUS_TOKEN`
